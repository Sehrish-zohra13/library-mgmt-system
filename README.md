Smart Library Inventory Management System

This project is a simple web-based library management system developed using Spring Boot for backend and HTML, CSS, JavaScript for frontend.

The system helps in managing books, users, and transactions like issuing and returning books. It replaces manual work with a digital system and makes the process faster and easier.

Features
Add books
View books
Add users
View users
Issue books
Return books
Track availability
